i = insert mode
o = insert on new line
p = paste on new line
y = yank (copy)
control v = visual block. You can highlight by characters (up, down, left/right)
shift v = highlights an entire line
control n = open and close nerd tree
;q = quit
;w = write
;wq = write quit
;q! = force quit
;wq! = force write quite
;vs = vertical split (put a space and you can tab into a new file)
control l = moves from your pane to the right pane
control h = moves from your pane to the left pane
control j = moves from your pane down
control k = moves from your pane up
j = moves down
k = moves up
l = moves right
h = moves left
shift a = moves cursor to end of line in INSERT mode
shift i = moves cursor to beginning of line in INSERT mode
w = moves cursor forward one whole word
b = moves cursor backward one whole word
u = undo 
control r = redo 
shift d = deletes from cursor to the end of line
dd = deletes an entire line
control u = page up
control d = page down
gg = goes to the top of file
shift g = goes to the bottom of file
